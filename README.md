## 작업
React App 생성

``` 
npx create-react-app egov-react
``` 

.env 파일 생성

``` 
NODE_PATH=src
```

src/Components 폴더 생성

``` 
yarn add prop-types
``` 

라우터 처리
```         
yarn add react-router-dom
``` 

CSS 추가하기 

``` 
yarn add styled-components
yarn add styled-reset
``` 

네트워킹 : HTTP request (API 호출)

``` 
yarn add axios
``` 

앱 실행 
``` 
yarn start
```
